##项目整体目录结构:##
	lirageProject :

				jquery-1.10.2.min.js jquery库

				lirange_mobile_reset.css  mobile端重置样式
			|-image :

			|-common
				headerjs-view.php 头部模块

		|-controller
		   index.php
